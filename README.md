# Dev.Portfolio | 20

这是一个基于 **Warm Futurist（温暖未来主义）** 审美设计的单页交互式个人作品集。该项目采用了 Stone-50 背景与 Orange-500 强调色的配色方案，旨在平衡逻辑的严谨感与视觉的温度。

---

## 🚀 核心特性

* **交互式 Hero 背景**：利用 HTML5 Canvas 渲染的粒子网络，支持鼠标避障交互。
* **多维数据可视化**：
    * **技能雷达图**：展示逻辑思维、UI设计、算法效率等六个维度的能力。
    * **技术栈构成图**：直观呈现 Python、HTML/CSS 和 JavaScript 的占比。
* **动态作品集过滤**：支持按分类（Python/AI、Web）实时筛选作品，并带有流畅的入场动画。
* **审美展示模块**：通过 JS Tab 切换显示音乐（Music）与电影（Movies）偏好。
* **全响应式设计**：基于 Tailwind CSS 构建，适配从桌面端到移动端的所有设备。

---

## 🛠️ 技术栈

* **样式库**：Tailwind CSS (CDN 版本)。
* **图表库**：Chart.js。
* **脚本逻辑**：原生 JavaScript (Vanilla JS)。
* **特效渲染**：HTML5 Canvas API。

---

## 📂 页面结构

1.  **导航栏 (Navigation)**：带毛玻璃效果（Glassmorphism）的固定式导航。
2.  **首屏 (Hero)**：动态粒子背景与核心价值宣言。
3.  **关于 (About)**：文字简介结合 Chart.js 数据可视化组件。
4.  **作品 (Portfolio)**：支持分类筛选的项目卡片网格。
5.  **审美 (Taste)**：个人兴趣、音乐循环列表与必看电影清单。
6.  **笔记 (Notes)**：思维碎片与外部链接列表。

---

## ⚙️ 快速自定义

* **修改作品信息**：在脚本部分的 `portfolioData` 数组中更新对象内容即可。
* **调整图表数据**：搜索 `new Chart` 并在 `data` 属性中修改数值。
* **更换主题色**：通过全局替换 Tailwind 类名（如将 `orange-500` 替换为其他色系）。
